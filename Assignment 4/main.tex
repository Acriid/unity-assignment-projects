\section*{What want to do}
Make an algorithm that makes levels. This include:
\begin{itemize}
    \item Entrance and Exit from room.
    \item Correct floor, door and wall placements.
    \item Navmesh of the room.
    \item A goal for the room. (item to get, item to drop)
\end{itemize}
\subsection*{Room Checkpoints}
\begin{itemize}
    \item Entrance is not blocked.
    \item Exit is not blocked.
    \item Floors are all connected.
    \item There are no walls on top of the floor
    \item The floor is a minimum of x by x and maximum of y by y
    \item All the walls are connected.
    \item The goal is accessible.
\end{itemize}
